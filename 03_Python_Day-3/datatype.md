- String : "Hello, World!", 'Python', """Triple quotes""", str(123) , b'a\x\01c'

**Example:**

```python
# Creating string variables
greeting = "Hello, World!"
name = 'Mayank'
multiline = """This is a
multiline string."""

print(greeting)      # Output: Hello, World!
print(name)          # Output: Mayank
print(multiline)     # Output: This is a
           #         multiline string.
```
